# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at that link. All participants in this project are expected to uphold this code.

## Reporting

If you experience or witness unacceptable behavior, please email **conduct@nxtg.ai**.

Reports will be reviewed and investigated promptly and fairly.
